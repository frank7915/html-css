    我们收集的个人数据

    Microsoft 收集数据以便高效运营，以及为您提供最佳的产品体验。 您直接提供部分此类数据，例如当您创建 Microsoft 帐户、向必应提交搜索查询、向 Cortana 发出语音指令、向 OneDrive 上载文件、购买 MSDN 订阅、注册 Office 365 或联系我们获得支持时。 我们通过记录您如何与我们的产品交互而获得部分数据，例如，通过 Cookie 等技术，以及从您设备上运行的软件接收错误报告或使用数据。 我们还从第三方获取数据。

    我们如何使用个人数据

    Microsoft 通过我们收集的数据向您提供我们的产品，产品内容包括使用数据改进并个性化您的体验。 我们也可能使用数据来与您通信，例如通知您有关您的帐户、安全性更新和产品信息。 我们也使用数据帮助显示更多相关的广告，无论是在我们自己的产品（如 MSN 和必应）中还是在第三方提供的产品中。 但是，我们不会使用您在电子邮件、聊天、视频通话或语音邮件中所说的内容或您的文档、照片或其他个人文件来针对您投放广告。

    • Cookie 和类似技术

    Microsoft 使用 Cookie（下载到您设备中的小型文本文件）和类似的技术来提供我们的网站和联机服务并协助收集数据。 Cookie 可让我们可以存储您的偏好和设置以及其他一些功能；使您能够登录；提供基于兴趣的广告；打击欺诈行为；分析我们的网站和联机服务的性能。 Microsoft 应用程序使用其他标识符，如 Windows 内的广告 ID，以实现类似目的。 我们也使用 Web 信号帮助提供 Cookie 和收集用法和性能数据。 我们的网站可能包含来自第三方服务提供商的 Web 信号和 Cookie。 您有各种可以控制 Cookie、Web 信号和类似技术的工具，包括用于阻止和删除 Cookie 的浏览器控件和来自某些第三方分析服务提供商的控件，以通过 Web 信号退出数据收集。 您的浏览器和其他选择可能影响您对我们产品的体验。

    • Cortana

    Cortana 是您的个人助理。 当您登录并允许 Cortana 使用来自您的设备、个人 Microsoft 帐户、其他 Microsoft 服务和您选择连接的第三方服务的数据时，Cortana 能更好地为您服务。 如果您选择不登录 Cortana，您仍然可以与 Cortana 聊天并使用 Cortana 帮助您搜索 Web 和您的设备。 但是，如果您不登录，您的体验将更加受限并且无法个性化。

    • Microsoft Health 服务

    Microsoft Health 服务可以帮助您了解和管理您的健康数据。 这些服务包括 Microsoft Band 设备、Microsoft Health 应用、HealthVault 和其他相关服务。 Band 帮助您跟踪心率和步行步数等数据。 Band 还可以使用 Cortana 做笔记，接收来自手机的通知。 Microsoft Health 应用将数据发送到 Microsoft 的服务器，并允许您管理和控制数据。 该应用为手机上的其他应用提供了向 Band 发送通知的渠道。 HealthVault 是一个个人健康平台，可让您在线收集、编辑、添加和存储健康数据。 通过 HealthVault，您还可以选择与家人、照顾者和医护人员共享您的健康数据。

    Microsoft Health 服务收集和使用您的数据来提供服务，其中包括改进和个性化您的体验。 您通过 Microsoft Health 服务向 Microsoft 提供的健康数据不会与来自其他 Microsoft 服务的数据合并，或在未经您明确同意的情况下用于其他用途。 例如，在未经您同意的情况下，Microsoft 不会使用您的健康记录数据向您进行推销或广告宣传。
